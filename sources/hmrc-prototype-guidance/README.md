# HMRC Prototype Guidance

This source package tracks authoring work for `hmrc-prototype-guidance`.

## Workflow

1. Update `site.yaml` with entry points, auth notes, and task ideas.
2. Run `./nexui survey-source <source-package>` to capture entry-point states.
3. Review captures and expand `task_ideas`.
4. Add recording recipes under `recipes/`.
5. Use `./nexui record` to generate benchmark task packages.
