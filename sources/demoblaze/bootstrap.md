# Bootstrap

Demoblaze is a public live demo and does not require local bootstrap for intake.

## Base URL

`https://demoblaze.com/`

## Commands

```bash
./nexui validate-source sources/demoblaze --check-remote
./nexui survey-source sources/demoblaze --overwrite
```

## Notes

- No login is required for the first planned task family.
- Prefer bounded add-to-cart and cart-review tasks over long purchase flows.
