# Reset Notes

## Statefulness

- The storefront is public and shared.
- Cart state can change during a session and should be verified before recording.

## Practical Reset Guidance

- Start new recordings from the storefront home page whenever possible.
- Use the cart page to confirm current state before recording cart-sensitive tasks.
- Prefer stop-before-purchase tasks to avoid unintended side effects.
